Pipeline

How the Model Works

A structured NLP and machine learning pipeline from raw text to personality classification.

01
Data Collection & Cleaning
Kaggle MBTI dataset with 8,600+ rows of user posts paired with personality labels. URLs, special characters, stop words, and noise are removed. Text is lowercased and normalized.
02
Feature Engineering
TF-IDF vectorization extracts word-frequency features. Additional features include post count per user, average post length, and frequency of type-related keywords in posts.
03
Model Training & Evaluation
Four classifiers — Logistic Regression, Random Forest, SVM, and XGBoost — are trained and evaluated using accuracy, F1-score, and confusion matrices for all 16 classes.

Performance

Model Comparison

Multiple classifiers were benchmarked. XGBoost achieved the best generalization on unseen samples.

Ensemble of decision trees. Provides feature importance rankings useful for understanding which words signal each type.
